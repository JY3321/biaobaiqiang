# 表白墙
简单实现表白墙项目，主要适配手机浏览器访问

# 体验地址
[chunge.6655.la](chunge.6655.la)

# 使用框架
springboot + mybatis-plus + Vue + iviewui + mescroll 

# 实现功能
* 登录、退出
* 分页查询，下拉刷新上拉加载
* 注册，邮箱激活，上传头像
* 发布信息
* 点赞
